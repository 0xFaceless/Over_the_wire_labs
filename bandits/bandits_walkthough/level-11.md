# Bandit Level 11 → 12

## Objective

Retrieve the password for Bandit Level 12. The password is stored in `data.txt`, where all lowercase (`a-z`) and uppercase (`A-Z`) letters have been rotated by 13 positions (ROT13).

**Username:** `bandit11`

## Skills Learned

- Understanding the ROT13 cipher
- Translating characters with `tr`
- Reading encoded text

## Commands Used

```bash
ssh
ls
cat
tr
```

## Key Takeaways

- ROT13 shifts each alphabetic character by 13 positions.
- The `tr` command translates characters from one set to another.
- Here strings (`<<<`) can be used to pass text directly to a command.
