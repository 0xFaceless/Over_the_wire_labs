# Bandit Level 2 → 3

## Objective

Retrieve the password for Bandit Level 3. The password is stored in a file named `--spaces in this filename--` in the home directory.

**Username:** `bandit2`

## Skills Learned

- Handling filenames with spaces
- Understanding the `--` option separator
- Reading files with special names

## Commands Used

```bash
ssh
ls
cat
```

## Key Takeaways

- Spaces in filenames should be enclosed in quotes.
- The `--` separator tells a command to stop parsing options.
- Combining quotes with `--` allows commands to access filenames that begin with `--`.
