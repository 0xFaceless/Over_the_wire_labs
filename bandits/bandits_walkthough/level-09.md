# Bandit Level 9 → 10

## Objective

Retrieve the password for Bandit Level 10. The password is stored in the file `data.txt` and is one of the few human-readable strings, preceded by several `=` characters.

**Username:** `bandit9`

## Skills Learned

- Identifying file types
- Extracting printable strings from binary files
- Searching text with `grep`
- Combining commands with pipes (`|`)

## Commands Used

```bash
ssh
ls
file
strings
grep
```

## Key Takeaways

- The `file` command identifies the type of a file.
- `strings` extracts printable text from binary files.
- `grep` filters output based on a pattern.
- Pipes (`|`) allow multiple commands to work together efficiently.
