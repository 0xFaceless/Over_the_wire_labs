# Bandit Level 5 → 6

## Objective

Retrieve the password for Bandit Level 6. The password is stored somewhere under the `inhere` directory and has the following properties:

- Human-readable
- 1033 bytes in size
- Not executable

**Username:** `bandit5`

## Skills Learned

- Searching files with `find`
- Filtering files by type, size, and permissions
- Reading file contents

## Commands Used

```bash
ssh
cd
ls
find
cat
```

## Key Takeaways

- The `find` command can search for files using multiple conditions.
- Combining filters makes locating files much easier than searching manually.
- `cat` is used to display the contents of the matching file.
