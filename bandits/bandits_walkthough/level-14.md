# Bandit Level 14 → 15

## Objective

Retrieve the password for Bandit Level 15. Submit the current level's password to a service listening on `localhost` at port `30000`.

**Username:** `bandit14`

## Skills Learned

- Understanding client-server communication
- Using Netcat (`nc`)
- Sending data through a TCP connection

## Commands Used

```bash
ssh
man
nc
echo
```

## Key Takeaways

- `nc` (Netcat) can connect to TCP and UDP services.
- `echo` sends text to standard output.
- Pipes (`|`) allow one command's output to become another command's input.
- Netcat is one of the most useful networking tools for Linux and cybersecurity.
