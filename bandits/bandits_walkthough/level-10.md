# Bandit Level 10 → 11

## Objective

Retrieve the password for Bandit Level 11. The password is stored in the file `data.txt`, which contains Base64 encoded data.

**Username:** `bandit10`

## Skills Learned

- Understanding Base64 encoding
- Decoding Base64 data
- Using the `base64` command

## Commands Used

```bash
ssh
ls
base64
```

## Key Takeaways

- Base64 is an encoding scheme, not encryption.
- The `base64 -d` option decodes Base64-encoded data.
- Reading command help pages is useful when learning new commands.
