# Bandit Level 3 → 4

## Objective

Retrieve the password for Bandit Level 4. The password is stored in a hidden file inside the `inhere` directory.

**Username:** `bandit3`

## Skills Learned

- Directory navigation
- Hidden files in Linux
- Using `ls -la`
- Reading file contents

## Commands Used

```bash
ssh
cd
ls
ls -la
cat
```

## Key Takeaways

- Files beginning with `.` are hidden by default.
- `ls -la` displays hidden files and additional file information.
- `cat` can be used to read the contents of hidden files.
